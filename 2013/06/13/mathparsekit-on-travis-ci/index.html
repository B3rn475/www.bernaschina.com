MathParseKit on Travis-CI

From today MathParseKit will be automatically tested by Travis-CI.

Travis CI is an online service for the Continuous Integration of GitHub repositories.

The main supported languages are:

  • C
  • C++
  • Clojure
  • Erlang
  • Go
  • Groovy
  • Haskell
  • Java
  • JavaScript
  • Perl
  • PHP
  • Python
  • Ruby
  • Scala
